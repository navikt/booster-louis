# Booster Louis

Presentasjon bygget med [Slidev](https://sli.dev/).

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Åpne [localhost:3030](http://localhost:3030) i nettleseren.

## Build

```bash
pnpm build
```

## Layouts

Tilgjengelige layouts:

- `cover` – Forsideslide med logo-slot
- `default` – Standard slide med header/content/footer
- `two-cols` – To kolonner med header
- `image-right` – Innhold til venstre, bilde til høyre
- `cta` – Call-to-action / avslutningsslide

## CSS-klasser

Se `style.css` for tilgjengelige utility-klasser som `.nais-card`, `.feature-card`, `.icon-badge`, `.callout`, `.chip`, `.gradient-bar` med mer.
